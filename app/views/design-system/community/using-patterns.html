Using the system

Components and some patterns come with code to make it easy for you to use them in your project.

There will soon be an updated version of the prototype kit with the NHSBSA design language for teams working on services.

Our deprecated prototype kit is still on GitHub, but no longer maintained

Using components in the prototype kit

Use the HTML we provide to use components in your prototypes. Simple copy the snippets into your prototype pages. Some components also need CSS and JavaScript to make them work. If this is the case, we also provide the code.

Sketch files

We are working on providing a sketch file with our patterns and components included.NHS.UK sketch files can be downloaded from GitHub.
