When to use this component

This component is a variation of the GOVUK checkboxes.

Use this component when the form is not the most important element on a page. Or when the size of the GOVUK radio buttons creates a false hierarchy of content.

A common use would be filtering search results.

When not to use this component

Always start with the GOVUK checkboxes. Don't use this component as part of a 'one thing per page' pattern.

User research

  • A number of services are using this pattern in the Home Office. From the research teams have done, no identified issues with the size or usability of the smaller checkboxes.
  • Component needs further research on mobile devices.

Services using this pattern

  • DAP
  • Resettlement services
  • General Aviation
  • ASRU
  • Application platform
  • Data services
