Patterns

Postcode lookup

Allow users to find an address using a postcode lookup.

Guidance

It can be used:

  • to find a single address
  • to add an address manually
  • as part of a loop to add multiple addresses

It should not be used:

  • if the address is needed as structured data

The postcode lookup pattern should:

  • replay the address options clearly
  • give the user flexibility to add an address in different ways

User research

  • Users living in new-build accomodation who knew their address wouldn't be found had no problem with free text input.
