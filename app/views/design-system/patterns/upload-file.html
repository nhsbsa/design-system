Add a file

An interaction that allows a user to add (upload) one or more files.

Add a single file

Add multiple files

Guidance

It can be used:

  • to add single files
  • to add multiple files
  • for a range of file types

It should not be used:

  • in some internal services where there is a need for bulk uploading

The Add file pattern should:

  • replay the file details clearly to reduce user error
  • show the progress of an upload so users know that something is happening
  • allow the user to skip the process if it is optional
